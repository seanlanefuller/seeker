# Seeker: A Journey of Wonder

*Wander. Discover. Wonder.*

**Seeker** is a minimal, atmospheric web-based exploration experience. Navigate a vast, procedurally accented void to discover fragments of ancient wisdom and cosmic truths.

## Features

- **Atmospheric Exploration**: Move through a serene vector-grid universe.
- **Wonders of Wisdom**: Discover "Wonders"—carefully curated quotes from philosophers and scientists.
- **Dynamic Radar System**: A pulse-based compass guides you toward the nearest undiscovered wonder.
- **Procedural Audio**: Features a deep-space ambient drone and interactive sound effects.
- **Void Anomalies**: Navigate around dark anomalies that populate the cosmic expanse.

## Controls

- **Movement**: Use `Arrow Keys` or `WASD` to navigate.
- **Interaction**: Follow the orange radar indicator to find Wonders. Colliding with a Wonder reveals its message.

## How to Run

Simply open `index.html` in any modern web browser to begin your journey.
